Refresh Dialog Fields

Refresh fields of a service dialog:

Request:

POST /api/service_dialogs/2
{
  "action" : "refresh_dialog_fields",
  "resource" : {
    "dialog_fields" : {
      "my_field" : "current value for my_field"
    },
    "fields" : [ "my_field" ]
  }
}
